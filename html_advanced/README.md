# HTML Advanced

This project focuses on learning HTML tags to structure a web page.

## Learning Objectives

- HTML5 guidelines and best practices
- Creating the skeleton of an HTML5 page
- Using semantic HTML tags (header, main, footer, article, nav, section, aside)
- Understanding div vs span use cases
- Following hierarchical order for headings
- Creating lists in HTML
- Understanding media differences (SVG, GIF, PNG, JPG)
- Structuring data in tables
- Integrating video and audio
- Embedding external content
